# 中国語 表現図鑑 — 運用マニュアル

このフォルダがサイト一式です。`index.html` は基本もう触りません。
ふだんの更新は **「videos.csv に1行足す」＋「slides/ にPNGを置く」** だけです。

---

## フォルダの中身

```
中国語表現図鑑/
├─ index.html      ← サイト本体（触らない）
├─ videos.csv      ← これだけ編集する（動画一覧データ）
├─ slides/         ← スライド画像（PNG）を入れるフォルダ
│   └─ sample-hen.png   ← 動作確認用サンプル（消してOK）
└─ README.md       ← この説明書
```

## ふだんの更新作業（動画を出したとき）

### 手順1｜videos.csv に1行足す

`videos.csv` をExcelやNumbers、テキストエディタで開いて、一番下に1行追加します。
列の順番は以下の通り：

| 列名 | 入れるもの | 例 |
|---|---|---|
| タイトル | 動画タイトル | 【中国語】〜の使い分け |
| キーワード | 扱った中国語を ` / ` で区切る | `很 / 非常 / 太〜了` |
| 要点 | ひとことメモ | 程度表現の使い分け。〜 |
| ジャンル | 使い分け / 文法 / 語彙 / 旅行 のどれか | 使い分け |
| 動画URL | YouTubeのURL | https://www.youtube.com/watch?v=... |
| 公開日 | YYYY-MM-DD | 2026-06-21 |
| 画像ファイル名 | slides/ に置くPNGの名前（空でもOK） | hen.png |

- **キーワードが空の行は、サイトに表示されません**（vlog・雑談回はここを空にすればOK）
- **画像ファイル名が空の行は、プレースホルダー（文字タイル）で表示**されます。あとからPNGを足してファイル名を書けば画像に変わります

### 手順2｜スライドPNGを slides/ に置く

Keynoteから書き出したPNGを `slides/` フォルダに入れて、
そのファイル名を videos.csv の「画像ファイル名」列に書きます。

> ファイル名は半角英数がおすすめ（例：`hen.png`, `le-2.png`）。
> 日本語・中国語のファイル名でも動きますが、英数の方がトラブルが少ないです。

### 手順3｜公開先にアップ

編集した `videos.csv` と、追加した `slides/○○.png` を、
公開先（GitHub Pages など）にアップロードします。これで反映完了。

## 自動でできること / 手動でやること（整理）

| 作業 | 自動 / 手動 | 補足 |
|---|---|---|
| タイトル・URL・公開日の取得 | （半自動も可） | YouTube APIで取れるが、初期設定が必要。週1まとめてなら手入力でも十分軽い |
| キーワードの抽出 | **手動** | 「この動画は何を扱ったか」の判断はあなたの価値 |
| 要点の一文 | **手動** | あなたの言葉で書くのがサイトの魅力 |
| ジャンル分け | **手動** | |
| スライドPNGの配置 | **手動** | Keynoteから書き出して slides/ へ |
| データ読み込み→サイト反映 | **自動** | index.html が videos.csv を読んで自動表示 |

ポイント：**機械的な情報は自動化できるが、中身の判断は手動**。
「週1でまとめて数行足す」運用なら、手入力でも負担は小さいです。

## リクエスト機能（解説してほしい表現の募集）

サイトには「リクエストを送る」ボタンが2か所あります：
- ページ下部の常設バナー
- 検索して0件だったときの誘導

### 送り先（Googleフォーム）の設定方法

1. Googleフォームを新規作成（無料・自分のGoogleアカウント）
   - 質問は「解説してほしい表現・知りたい違い」など自由記述ひとつでOK
2. 右上「送信」→ リンクアイコン → URLをコピー
3. `index.html` を開き、上の方にあるこの行を書き換える：

```js
const REQUEST_FORM_URL = "#";
```
↓ コピーしたURLを貼る

```js
const REQUEST_FORM_URL = "https://forms.gle/xxxxxxxx";
```

4. 保存してアップ。これでボタンがフォームにつながります。

### 通知（誰かが送ったら気づくには）

初期状態ではメール通知は来ません。届くようにするには：
- フォーム編集画面 → 「回答」タブ → 右上の ⋮ メニュー
- 「新しい回答についてのメール通知を受け取る」にチェック

これでリクエストが来るたびにGmailに通知が届きます。

## Macで今すぐ見た目を確認したいとき

`index.html` を**ダブルクリックで開くと、videos.csv を読み込めません**
（ブラウザのセキュリティ制限。これは正常です）。

確認したいときは、このフォルダで簡易サーバーを立てます：

1. ターミナルを開く
2. このフォルダに移動（`cd ` のあとにフォルダをドラッグ＆ドロップ → Enter）
3. 次を実行：
   ```
   python3 -m http.server 8000
   ```
4. ブラウザで `http://localhost:8000` を開く
5. 確認が終わったら、ターミナルで Ctrl+C で止める

※ iPhoneでの確認は、次の「公開」をしてからになります。

## 困ったとき

- **カードが表示されない** → サーバー経由で開いているか確認（上の「Macで確認」参照）。`videos.csv` が同じフォルダにあるか確認
- **画像が出ない** → ファイル名がCSVの「画像ファイル名」列と一致しているか、`slides/` に入っているか確認（大文字・小文字も区別されます）
- **特定の動画が出ない** → キーワード列が空欄になっていないか確認
